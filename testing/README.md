# Testing 

This folder contains documentation related to the testing and evaluation of the Study Zen, a Stress & Study Balance Tracker prototype. In addition to this, usability testing was carried out to test whether the prototype works effectively and whether it is easy to use. Some students were used as test users for the usability testing. They were asked to test the system by carrying out various activities such as logging into the system, recording their study hours, tracking their stress levels, accessing relaxation tips, and accessing the emergency hotline numbers. During the testing of the system, the test users interacted with the system. The findings revealed that the system was easy to understand and the test users found the features useful for managing their studies and stress levels. The files in this folder document the testing process, user feedback, and observations gathered during the usability testing stage.
